This is an [Astro](https://astro.build) project for Haruki Kuriwada's personal website.

## Getting Started

Run the development server:

```bash
npm run dev
```

Open [http://localhost:4321](http://localhost:4321) with your browser to see the result.

The site entrypoint is `src/pages/index.astro`.

## Scripts

```bash
npm run dev
npm run build
npm run start
npm run lint
npm run typecheck
```
